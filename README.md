# Predizer

**Predizer** é uma plataforma gamificada de pesquisa e previsão: você cria/participa de mercados sobre eventos futuros (preços, adoção, volume etc.), aloca um valor em cada palpite e acompanha desempenho e resultados em painéis analíticos.

> Projeto em modo **MVP/Demo**: as funcionalidades principais estão prontas para apresentação e evolução para backend real (API + banco) quando necessário.

---

## O que o app faz

- **Mercados de previsão:** criar, listar e explorar mercados por categorias, buscas e filtros.
- **Negociação SIM/NÃO (demo):** simula compra de posições com saldo de carteira.
- **Carteira (demo):** saldo, depósitos/saques simulados e histórico de transações.
- **Portfólio:** posições, total investido e métricas de performance (MVP).
- **Admin Hub:** visão de métricas, gestão de mercados e ações administrativas.
- **UI completa:** navegação e telas prontas para integração com backend real.
